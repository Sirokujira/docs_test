template<typename PointT, typename PointNT>
class SampleConsensusModelFromNormals¶

SampleConsensusModelFromNormals represents the base model class for models that require the use of surface normals for estimation.

Subclassed by pcl::SampleConsensusModelCone< PointT, PointNT >, pcl::SampleConsensusModelCylinder< PointT, PointNT >, pcl::SampleConsensusModelNormalPlane< PointT, PointNT >, pcl::SampleConsensusModelNormalSphere< PointT, PointNT >
