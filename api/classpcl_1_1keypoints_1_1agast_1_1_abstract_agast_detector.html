class AbstractAgastDetector¶

Abstract detector class for AGAST corner point detectors.

Adapted from the C++ implementation of Elmar Mair (http://www6.in.tum.de/Main/ResearchAgast).

Author

Stefan Holzer

Subclassed by pcl::keypoints::agast::AgastDetector5_8, pcl::keypoints::agast::AgastDetector7_12s, pcl::keypoints::agast::OastDetector9_16
