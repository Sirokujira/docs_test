integer starting from 1

Float going from 0.0 (transparent) to 1.0 (opaque)

PCL_VISUALIZER_LINE_WIDTH¶

Integer starting from 1.

3 floats (R, G, B) going from 0.0 (dark) to 1.0 (light)
