template<typename PointT>
class Comparator¶

Comparator is the base class for comparators that compare two points given some function.

Currently intended for use with OrganizedConnectedComponentSegmentation

Author

Alex Trevor

Subclassed by pcl::experimental::EuclideanClusterComparator< PointT, PointLT >, pcl::GroundPlaneComparator< PointT, PointNT >, pcl::PlaneCoefficientComparator< PointT, PointNT >
