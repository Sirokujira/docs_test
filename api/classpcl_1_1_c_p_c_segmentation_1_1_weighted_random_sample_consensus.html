class WeightedRandomSampleConsensus : public pcl::SampleConsensus<WeightSACPointType>¶

WeightedRandomSampleConsensus represents an implementation of the Directionally Weighted RANSAC algorithm, as described in: “Constrained Planar Cuts - Part Segmentation for Point Clouds”, CVPR 2015, M.

Schoeler, J. Papon, F. Wörgötter.

Note

It only uses points with a weight > 0 for the model calculation, but uses all points for the evaluation (scoring of the model) Only use in conjunction with sac_model_plane If you use this in a scientific work please cite the following paper: M. Schoeler, J. Papon, F. Woergoetter Constrained Planar Cuts - Object Partitioning for Point Clouds In Proceedings of the IEEE Conference on Computer Vision and Pattern Recognition (CVPR) 2015

Author

Markus Schoeler (mschoeler@web.de)
