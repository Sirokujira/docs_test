Class Documentation¶

template<typename PointT, typename PointNT>
class PlaneCoefficientComparator : public pcl::Comparator<PointT>¶

PlaneCoefficientComparator is a Comparator that operates on plane coefficients, for use in planar segmentation.

In conjunction with OrganizedConnectedComponentSegmentation, this allows planes to be segmented from organized data.

Author

Alex Trevor

Subclassed by pcl::EdgeAwarePlaneComparator< PointT, PointNT >, pcl::EuclideanPlaneCoefficientComparator< PointT, PointNT >, pcl::PlaneRefinementComparator< PointT, PointNT, PointLT >, pcl::RGBPlaneCoefficientComparator< PointT, PointNT >
