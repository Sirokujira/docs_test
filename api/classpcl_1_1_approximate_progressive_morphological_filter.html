template<typename PointT>
class ApproximateProgressiveMorphologicalFilter : public pcl::PCLBase<PointT>¶

Implements the Progressive Morphological Filter for segmentation of ground points.

Description can be found in the article “A Progressive Morphological Filter for Removing Nonground Measurements from

Airborne LIDAR Data” by K. Zhang, S. Chen, D. Whitman, M. Shyu, J. Yan, and C. Zhang.
