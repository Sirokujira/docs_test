class PCLBase¶

PCL base class.

Implements methods that are used by most PCL algorithms.

Subclassed by pcl::Feature< PointT, PointFeature >, pcl::Keypoint< PointT, PointT >, pcl::ApproximateProgressiveMorphologicalFilter< PointT >, pcl::ConditionalEuclideanClustering< PointT >, pcl::EuclideanClusterExtraction< PointT >, pcl::ExtractPolygonalPrismData< PointT >, pcl::Filter< PointT >, pcl::GrabCut< PointT >, pcl::LabeledEuclideanClusterExtraction< PointT >, pcl::MomentOfInertiaEstimation< PointT >, pcl::Morphology< PointT >, pcl::OrganizedConnectedComponentSegmentation< PointT, PointLT >, pcl::OrganizedEdgeBase< PointT, PointLT >, pcl::OrganizedMultiPlaneSegmentation< PointT, PointNT, PointLT >, pcl::PCA< PointT >, pcl::ProgressiveMorphologicalFilter< PointT >, pcl::RegionGrowing< PointT, NormalT >, pcl::registration::ELCH< PointT >, pcl::SACSegmentation< PointT >, pcl::SegmentDifferences< PointT >, pcl::StatisticalMultiscaleInterestRegionExtraction< PointT >, pcl::SupervoxelClustering< PointT >, pcl::SurfelSmoothing< PointT, PointNT >, pcl::Registration< PointT, PointT >, pcl::Registration< PointT, PointT, Scalar >

Protected Functions

bool initCompute()

This method should get called before starting the actual computation.

Internally, initCompute() does the following:

  • checks if an input dataset is given, and returns false otherwise

  • checks whether a set of input indices has been given. Returns true if yes.

  • if no input indices have been given, a fake set is created, which will be used until:

    • either a new set is given via setIndices(), or

    • a new cloud is given that has a different set of points. This will trigger an update on the set of fake indices

bool deinitCompute()

This method should get called after finishing the actual computation.
