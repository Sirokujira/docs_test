Class Documentation¶

template<typename PointT>
class Filter : public pcl::PCLBase<PointT>¶

Filter represents the base filter class.

All filters must inherit from this interface.

Author

Radu B. Rusu

Subclassed by pcl::ApproximateVoxelGrid< PointT >, pcl::BilateralFilter< PointT >, pcl::ConditionalRemoval< PointT >, pcl::Convolution< PointT >, pcl::FastBilateralFilter< PointT >, pcl::FilterIndices< PointT >, pcl::MedianFilter< PointT >, pcl::ProjectInliers< PointT >, pcl::SamplingSurfaceNormal< PointT >, pcl::UniformSampling< PointT >, pcl::VoxelGrid< PointT >
