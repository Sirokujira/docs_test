class DeviceMemory¶

DeviceMemory class

Note

This is a BLOB container class with reference counting for GPU memory.

Author

Anatoly Baksheev

Subclassed by pcl::gpu::DeviceArray< curandState >, pcl::gpu::DeviceArray< double >, pcl::gpu::DeviceArray< float >, pcl::gpu::DeviceArray< float4 >, pcl::gpu::DeviceArray< int >, pcl::gpu::DeviceArray< Label >, pcl::gpu::DeviceArray< NormalType >, pcl::gpu::DeviceArray< pcl::gpu::people::trees::Node >, pcl::gpu::DeviceArray< pcl::PointXYZ >, pcl::gpu::DeviceArray< pcl::tracking::ParticleXYZRPY >, pcl::gpu::DeviceArray< PointType >, pcl::gpu::DeviceArray< uint64_type >, pcl::gpu::DeviceArray< unsigned char >, pcl::gpu::DeviceArray< T >

Assignment operator.

void create(size_t sizeBytes_arg)¶

Allocates internal buffer in GPU memory.

If internal buffer was created before the function recreates it with new size. If new and old sizes are equal it does nothing.

Parameters

  • sizeBytes_arg: buffer size

void release()¶

Decrements reference counter and releases internal buffer if needed.

void copyTo(DeviceMemory &other) const¶

Performs data copying.

If destination size differs it will be reallocated.

Parameters

  • other: destination container

void upload(const void *host_ptr_arg, size_t sizeBytes_arg)¶

Uploads data to internal buffer in GPU memory.

It calls create() inside to ensure that intenal buffer size is enough.

Parameters

  • host_ptr_arg: pointer to buffer to upload

  • sizeBytes_arg: buffer size

void download(void *host_ptr_arg) const¶

Downloads data from internal buffer to CPU memory.
