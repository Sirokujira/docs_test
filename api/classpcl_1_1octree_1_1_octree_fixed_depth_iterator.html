template<typename OctreeT>
class OctreeFixedDepthIterator : public pcl::octree::OctreeBreadthFirstIterator<OctreeT>¶

Octree iterator class

Note

Iterator over all existing nodes at a given depth. It walks across an octree in a breadth-first manner.

Author

Fabien Rozar (fabien.rozar@gmail.com)
